# Ayoub El Maalmi — Portfolio

Innovative portfolio site built with Next.js 14, Tailwind CSS, and Framer Motion to showcase Ayoub El Maalmi’s AI, cloud, and full-stack projects.

## Stack

- Next.js 14 (App Router, TypeScript)
- Tailwind CSS with custom theming + glassmorphism accents
- Framer Motion micro-interactions & bespoke SVG data visualisation
- Lucide React icons
- React Hook Form contact launcher

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the site locally.

## Structure

- `app/` – Next.js app routes.
- `components/` – UI building blocks (Hero, Projects, Experience, etc.).
- `lib/data.ts` – Centralised content (experiences, projects, articles, stats).
- `public/` – Static assets (profile pictures, favicons, etc. — add your own).

## Deployment

Deploy seamlessly to Vercel or Netlify:

- Ensure environment uses Node.js 18+.
- Run the build step `npm run build`.
- Configure custom domain (e.g., `ayoub.elmaalmi.com`).

## Customisation

Update `lib/data.ts` to refresh content (projects, experience, stats, certifications).
